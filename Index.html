Web Security

How would you advise a potential client to secure his/her website from web security attacks? Note: the word count for your answer should not be greater than 250 words

[Adopt a cybersecurity framework, train your staff and keep them up to date with the system and all the changes, add HHTPS and SSL certificate, use anti-malware software and plugins and keep them up to date, choose a strong password, use a secure web-host that offers SFTP, backup your website regularly, change your CMS default settings because the most common attacks against websites are entirely automated and what many attack bots rely on is for users to have their CMS settings on default, implement an automatic user timeout, know your web server configuration files, make sure you apply for a web application firewall, add a two-factor authenticator, Record User Access and Administrative Privileges]
